# Backend setup
```
python -m venv <NAME>
```
```
pip install -r requirements.txt
```
```
python manage.py runserver
```

# Frontend setup
```
npm install
```
```
npm run dev
```

# Create super user for local admin 
```
python manage.py createsuperuser
```

# How to make migrations
```
python manage.py makemigration
```
```
python manage.py migrate
```
